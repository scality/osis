# Capture

Ride-along artifacts from manual lab runs. Used to build and maintain `scripts/*.sh`. Scrub passwords and internal IPs before committing.

- `SHELL_HISTORY.md` — every command run during the manual ride-along, in order, with outputs and timings.
- `DOC_GAPS.md` — deltas from the Confluence source doc.
- `aws-notes.md` — AMI IDs, region quirks, instance behavior.
- `troubleshooting.md` — recovery recipes for known failure modes.
